# Spotify AI DJ — Product Strategy Teardown

A standalone, responsive HTML portfolio case study by Aishwarya Chourasia.

## What is included

- Problem framing and Jobs to Be Done
- Public user-signal synthesis
- Current and proposed experience flows
- Metrics tree and custom North Star
- Interactive Discovery Dial prototype
- Proposed recommendation-system architecture
- Ranking logic and signal qualification
- Event instrumentation plan
- A/B/n experiment design
- Interactive impact model
- RICE prioritization
- Responsible AI, privacy, reliability, accessibility, and failure handling
- Six-week delivery roadmap
- Interview-ready narrative and source list

## Run locally

Open `index.html` in any modern browser. No build tools or external JavaScript libraries are required.

## Notes

This is independent portfolio work and is not affiliated with Spotify. Public facts are sourced in the website. Proposed UX, metrics, architecture, weights, impact assumptions, and delivery plans are illustrative.
