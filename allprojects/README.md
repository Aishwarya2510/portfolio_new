# All Projects Source Folder

Use this folder as the raw source library for rebuilding project pages.

Create one folder per project using a short slug-style name:

```text
allprojects/
  my-project-name/
    images/
    architecture-diagrams/
    raw-content/
    notes.md
```

Suggested use:

- `images/`: screenshots, product images, mockups, GIFs, and visual assets.
- `architecture-diagrams/`: system diagrams, data flows, process maps, Mermaid exports, or architecture visuals.
- `raw-content/`: drafts, copied notes, case-study text, PDFs, research notes, metrics, and anything I should reference.
- `notes.md`: a short project brief with title, timeline, role, team, goal, impact, links, and any instructions.

Copy `_template` when starting a new project folder.
