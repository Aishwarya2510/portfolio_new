# Project Momentum — Duolingo D7 Retention Experiment

A standalone, portfolio-ready product growth experiment case study.

## Contents
- `index.html`: Complete responsive website with no build step
- Experiment portfolio: Momentum Plan, Rescue Lesson, Buddy Sprint
- Funnel, metric tree, RICE prioritization, architecture, event taxonomy
- Interactive prototype, sample-size calculator, impact calculator, and decision simulator
- Risk, ethics, rollout plan, interview story, and source register

## Use
Open `index.html` in any modern browser. The page is responsive and print-ready.

## Important
This is an independent, pre-launch portfolio case study. Company-reported facts are linked in the source section. Proposed baselines, experiment effects, traffic, sample sizes, and RICE scores are planning assumptions, not Duolingo results.
